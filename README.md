# 盖洛普才干应用工具

一个静态网页工具，用于按才干名称查看 34 个盖洛普才干解读卡页面，并支持录入已完成测试后的前五 / 前十才干。

## 功能

- 34 个才干按四大领域分类展示
- 执行力：紫色
- 影响力：橙色
- 关系建立：蓝色
- 战略思维：绿色
- 输入才干标题后展示对应 PDF 页面图片
- 支持打开对应单页 PDF
- 支持“已做测试 / 没做测试”状态切换
- 已做测试时支持录入前五或前十才干
- 支持用户名注册 / 进入，并按用户名在浏览器本地记住已输入才干

## 本地预览

```bash
python3 -m http.server 8017
```

然后访问：

```text
http://127.0.0.1:8017/
```
